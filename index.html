API Service

To check the ranking of a specific package, please use the API as below 
http://R-Ranking.com:8000/downloads?package=package-name
For example, to check the ranking of package "Rcpp", we can enter the link below into any browser. 
http://R-Ranking.com:8000/downloads?package=Rcpp
